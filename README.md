# Local Installation

This section is for testing or building the app locally.

backend

1. cd backend
2. npm i
3. npm start

frontend

1. cd frontend
2. npm i
3. npm run dev

## Prerequisite

Please make sure that [Node.js](https://nodejs.org/en/) is installed on your system. The developers used v23.x
